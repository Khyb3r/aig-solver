# Non-Clausal AIG SAT Solver
This is a Non-Clausal SAT Solver for the AIG representation.

## File Structure

- /src contains all Solver source code.
- /Benchmarks contains all three sets of benchmarks all with their corresponding '.aig' files.
- /Scripts contains all the Python scriptsf for testing and experimentation.
- /logs contains raw log files and a .json per run on a selected set of benchmarks.
- /cadical-cnf-conversions contains '.cnf' files converted for their correspondong '.aig' files.


## Setup and Build

### Dependencies 
This solver relies on the AIGER library ensure you have it installed to run the solver.
